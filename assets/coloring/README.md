# Dibujos para colorear

Copia aquí **cualquier imagen** y la app la detecta sola.

## Formatos admitidos

| Formato | Extensiones      | Notas                                      |
|---------|------------------|--------------------------------------------|
| JPEG    | `.jpg` `.jpeg`   | Ideal                                      |
| PNG     | `.png`           | Ideal (mejor si el fondo es transparente)  |
| GIF     | `.gif`           | Soportado                                  |
| WebP    | `.webp`          | Soportado                                  |
| BMP     | `.bmp`           | Soportado                                  |
| AVIF    | `.avif`          | Puede fallar en algunos dispositivos       |

Recomendado: **líneas negras** sobre fondo **blanco**.

## Pasos

1. Copia el archivo en esta carpeta (no hace falta renombrar).
2. En la terminal:
   ```bash
   flutter pub get
   flutter run -d chrome
   ```
3. Abre **Colorea y Pinta** → tus dibujos aparecen solos.

No hace falta editar código ni registrar el nombre del archivo.
